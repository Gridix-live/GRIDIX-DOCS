Приоритет простой: чем выше в списке — тем важнее для первых пользователей.

## Технические правила для скриншотов

- Разрешение: **минимум 1440×900**, лучше 1920×1080
- Формат: **PNG** или **WebP** (не JPEG для интерфейса — артефакты)
- Где хранить: `/images/<раздел>/имя-файла.png`, например `/images/projects/lot-statuses-table.png`
- Имена файлов: только латиница, строчные, дефисы — никаких кириллических названий
- Тёмная/светлая тема: снимать в **светлой теме** (дефолт для Mintlify)
- Данные: использовать тестовый аккаунт, не реальные данные клиентов

## Правила для видео

- Длина: **60–90 секунд** на ролик, не больше
- Формат: **MP4 H.264**, или загружать на YouTube/Vimeo и встраивать через `<iframe>`
- Где хранить локально: `/images/videos/имя.mp4`
- Где встраивать в MDX: через `<iframe src="https://www.youtube.com/embed/ID" className="w-full aspect-video rounded-xl" />`
- Аудио: необязательно — можно субтитрами или без звука

## Как добавить скриншот в статью

Найди в нужном `.mdx`-файле комментарий вида:

```
{/* SCREENSHOT: что снять */}
```

Сразу после него добавь:

```mdx
<Frame caption="Короткое описание">
  <img src="/images/раздел/имя-файла.png" alt="Описание" />
</Frame>
```

---

## Как добавить видео

```mdx
<iframe
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="Название видео"
  frameborder="0"
  className="w-full aspect-video rounded-xl"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
/>
```
