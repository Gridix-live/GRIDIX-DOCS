The Agent Network lets you manage partner agencies and individual brokers who sell your properties. Track their performance, handle commissions, and manage approvals — all from one place.

<Note>
Agent Network is available on the Pro plan. Access it from the sidebar in the admin dashboard.
</Note>

<Note>
  **Screenshot:** Agent Network overview with stats cards and partner table
</Note>

## Dashboard

The Agent Network page shows key metrics at the top:

- **Total partners** — all registered partners
- **Active partners** — currently active and selling
- **Sales volume** — total revenue through agents
- **Pending requests** — applications awaiting review
- **Pending commissions** — unpaid commission amounts

## Partners List

Below the stats, you'll see a searchable and filterable table of all your partners.

### Filtering

<Note>
  **Screenshot:** Partner filters panel (status, type, commission range, date)
</Note>

Filter partners by:

- **Status** — all, pending, needs correction, active, blocked
- **Type** — all, agency, individual
- **Commission range** — min/max percentage
- **Date range** — when they joined

### Actions

From the partners list, you can:

- **Invite a new partner** — send an invitation link
- **View details** — open partner drawer (see [Agency Management](/en/agent-network/agencies))
- **Approve/Reject** — process pending applications
- **Mark as paid** — confirm commission payouts
- **Delete** — remove a partner

## Status Workflow

Partners go through these statuses:

| Status | Description |
|--------|-------------|
| Pending | New application, awaiting your review |
| Needs Correction | Application returned with feedback — agent must resubmit |
| Active | Approved and actively selling |
| Blocked | Deactivated — no longer accepting leads |

## General Conditions

The second tab on the Agent Network page lets you manage general terms and agreements for your agent program:

- Define partnership rules
- Upload or create agreements
- Manage digital signatures
- Import terms from Google Docs
