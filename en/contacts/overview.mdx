The Contacts section provides a unified view of all your clients (from leads) and agents in one searchable list. Access it from the sidebar in the admin dashboard.

<Note>
  **Screenshot:** Contacts list with filter buttons and type badges
</Note>

## What Are Contacts?

Contacts are automatically created from two sources:

- **Clients** — aggregated from your leads. Multiple leads from the same person (matched by email or phone) are grouped into a single contact entry showing the total lead count.
- **Agents** — from agent applications in your Agent Network. Shows agent status and type.

<Note>
Contacts are not created manually. They are generated automatically from leads and agent applications.
</Note>

## Viewing Contacts

The contacts list shows:

| Column | Description |
|--------|-------------|
| Name | Contact name |
| Type | Client or Agent badge |
| Email | Email address |
| Phone | Phone number |
| Details | Lead count (for clients) or agent status (for agents) |
| Date | When the contact was first created |

## Filtering

Use the filter buttons at the top to switch between:

- **All** — show all contacts
- **Clients** — show only lead-based contacts
- **Agents** — show only agent contacts

## Search

Search contacts by name, email, or phone number. The search filters the list in real time.

## Export

Export your contacts list as a CSV file. The export includes: contact name, type, phone, email, details, and date.

## How Deduplication Works

When multiple leads share the same email address or phone number, they are merged into a single contact. The contact entry shows the total number of leads from that person. This helps you see unique clients rather than individual form submissions.
