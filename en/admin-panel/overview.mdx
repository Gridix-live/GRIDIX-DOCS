The admin panel is where you manage projects, leads, team members, agents, and integrations. After logging in, you land here automatically.

<Note>
  **Screenshot:** Admin panel — sidebar navigation and main content area
</Note>

## Sidebar Navigation

The left sidebar contains all main sections:

- **Projects** — create and manage real estate projects
- **Leads** — track incoming leads from widgets
- **Contacts** — unified view of all clients and agents
- **Agent Network** — manage partner agencies and commissions
- **Subscription** — view and manage your plan
- **Widgets** — generate embed codes for your website
- **Analytics** — view project and lead performance metrics
- **Settings** — company profile, team members, notifications, branding

## Content Area

The main area displays the selected section. It adapts to show project lists, lead tables, settings forms, widget configuration, and more.

## Top Bar

The top bar includes:
- **Language toggle** — switch interface language
- **User menu** — profile and logout
- **Support** — quick access to help

## User Roles

<Tabs>
  <Tab title="Developer">
    Full access to all features:
    - Create and manage projects
    - Access all leads and contacts
    - Manage team members and agents
    - Configure integrations and subscriptions
  </Tab>
  <Tab title="Manager">
    Limited access based on assigned projects:
    - View and edit assigned projects only
    - Access leads from assigned projects
    - Cannot manage subscriptions, settings, or team members
  </Tab>
</Tabs>

## Language Support

The admin panel is available in 6 languages:
- English (en)
- Russian (ru)
- Georgian (ka)
- Arabic (ar)
- Hebrew (he)
- Turkish (tr)

Language preference is saved per user and persists across sessions.

## Mobile Access

The admin panel is fully responsive and works on desktop browsers, tablets, and mobile phones. The sidebar collapses into a mobile menu on smaller screens.
