В кабинете агента есть раздел аналитики, который показывает личную статистику: сколько заявок вы передали девелоперам, по каким проектам, на каком этапе они находятся и какие комиссии начислены.

**Что доступно:**
- количество переданных заявок по проектам;
- стадии заявок в воронке девелопера (если девелопер открыл эту информацию);
- источники лидов — откуда пришли клиенты;
- история комиссий: ожидает, начислено, выплачено.

Аналитика помогает понять, какие проекты приносят больше закрытых сделок, и планировать работу с клиентами эффективнее.
