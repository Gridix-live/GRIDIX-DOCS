Эта страница помогает команде, клиентам, брокерам и партнёрам говорить одним языком.

| Термин | Значение |
| --- | --- |
| GRIDIX | Цифровая платформа для управления проектами, лотами, заявками, CRM, публичным каталогом и брокерской сетью девелопера |
| Девелопер | Компания, которая управляет проектами, лотами, ценами, заявками, командой и брокерской сетью |
| Кабинет девелопера | Рабочее пространство компании в GRIDIX |
| Проект | Жилой комплекс, апарт-комплекс, коммерческий объект или другой проект девелопера |
| Лот | Конкретный объект продажи внутри проекта: квартира, апартамент, офис, помещение, вилла, таунхаус или другой объект |
| Смарт-каталог | Цифровой каталог лотов с актуальными ценами, статусами, характеристиками, планировками и материалами |
| Шахматка | Визуальный способ отображения лотов внутри смарт-каталога |
| Генплан | Визуальный план проекта с корпусами, объектами или зонами, через который пользователь может перейти к конкретному подпроекту |
| Подпроект | Корпус, башня, очередь, объект или отдельная часть проекта внутри генплана |
| Публичный каталог проекта | Публичная страница или интерфейс проекта, где клиент видит лоты, цены, статусы, планировки и форму заявки |
| CRM | Раздел для работы с заявками, контактами, ответственными, статусами и историей клиента |
| Лид | Заявка или потенциальный клиент, который проявил интерес к проекту, лоту или предложению |
| Источник заявки | Канал или пользователь, через которого пришла заявка: сайт, виджет, менеджер, брокер, агентство или другой источник |
| Брокерская сеть | Модуль, который помогает девелоперу подключать внешних брокеров и агентства к своим проектам |
| Партнёрская программа GRIDIX | Программа для участников, которые рекомендуют платформу GRIDIX новым клиентам или помогают с внедрением |
| Реферальный партнёр | Участник партнёрской программы, который передаёт качественные контакты потенциальных клиентов GRIDIX |
| Интегратор GRIDIX | Участник партнёрской программы, который помогает клиенту с внедрением, настройкой и сопровождением GRIDIX |
