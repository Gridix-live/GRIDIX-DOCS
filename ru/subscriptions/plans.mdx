GRIDIX поддерживает разные уровни возможностей. Точный набор функций зависит от тарифа и подключенных модулей.

## Что важно сравнить

| Возможность | Базовый сценарий | Расширенный сценарий |
|---|---|---|
| Проекты и лоты | Да | Да |
| Интерактивные планировки | Да | Да |
| Сбор лидов | Да | Да |
| CRM-интеграции | Может быть ограничено | Да |
| Кастомные домены | Может быть ограничено | Да |
| Брокерская сеть | Обычно расширенная функция | Да |
| Аналитика | Базовая | Расширенная |

<Note>
  Для актуальных условий тарифа напишите в поддержку: [@gridix\_bot](https://t.me/gridix_bot)
</Note>

## Когда нужен расширенный сценарий

- нужно подключить CRM;
- нужен кастомный домен;
- вы работаете с брокерской сетью;
- нужна расширенная аналитика;
- несколько менеджеров работают с разными проектами.
