Перед запуском GRIDIX подготовьте данные, которые понадобятся для проекта, публичного каталога, CRM, виджета сайта и команды.

## Основной список

- список проектов;
- список лотов;
- цены;
- статусы лотов;
- планировки;
- изображения и материалы проекта;
- характеристики лотов;
- данные менеджеров;
- роли и доступы команды;
- правила обработки заявок;
- список брокеров или агентств, если нужно подключить брокерскую сеть;
- данные для сайта, если планируется установка виджета;
- материалы для публичного каталога проекта.

## Для таблицы с лотами

Минимально подготовьте:

- номер лота;
- этаж, если проект относится к типу “Здание”;
- площадь;
- цену;
- статус;
- количество комнат, если применимо;
- дополнительные характеристики, которые нужно показывать клиенту.

## Пример структуры таблицы

| Колонка | Пример | Комментарий |
| --- | --- | --- |
| lot_number | A-1204 | Номер или внутренний код лота |
| floor | 12 | Нужен для типа “Здание” |
| area | 74.5 | Площадь в выбранной единице |
| rooms | 2 | Если применимо |
| price | 185000 | Цена без форматирования текстом |
| status | available | Значение нужно сопоставить со статусом GRIDIX |
| layout_url | ссылка или имя файла | Планировка или материал лота |

## Для сайта и виджета

Подготовьте:

- страницу сайта, куда будет встроен виджет;
- доступ к CMS или разработчику сайта;
- язык по умолчанию;
- проект или подпроект, который нужно показать;
- понимание, нужен ли полный проект, список лотов, шахматка, генплан или отдельный корпус.

## Для CRM

Подготовьте:

- какую CRM подключаете: AmoCRM или Bitrix24;
- доступ администратора CRM;
- воронку и этапы;
- правила назначения ответственных;
- какие поля заявки должны попадать в CRM.
