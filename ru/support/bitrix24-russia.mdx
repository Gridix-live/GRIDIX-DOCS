Прямое подключение Bitrix24 через OAuth работает для международных аккаунтов. Российские аккаунты Bitrix24 имеют ограничения на OAuth-авторизацию с зарубежными сервисами — поэтому стандартная кнопка «Подключить Bitrix24» может не сработать.

## Как подключить Bitrix24 Россия

Для российских аккаунтов интеграция настраивается через **Webhook**. Это занимает чуть больше времени, но работает стабильно.

Напишите в поддержку: [@gridix_bot](https://t.me/gridix_bot) — команда GRIDIX пришлёт пошаговую инструкцию с учётом вашей версии Bitrix24.

**Что нужно подготовить:**
- доступ администратора в Bitrix24;
- понимание, в какую воронку должны попадать лиды;
- контакт вашего специалиста по Bitrix24 (если настройкой занимается другой человек).

<Note>
  Для интеграторов: инструкция по webhook-подключению Bitrix24 доступна через поддержку. Обычная настройка через Webhook занимает 30–60 минут.
</Note>
