Пользовательские поля помогают хранить дополнительные параметры, которых нет в стандартной модели GRIDIX.

## Когда использовать

- вид из окна;
- корпус или секция;
- отделка;
- сторона света;
- специальные условия;
- внутренний ID из CRM или ERP.

## Типы полей

| Тип | Пример |
|---|---|
| Текст | "Секция A" |
| Число | "12" |
| Да/нет | "С отделкой" |
| Список | "Вид на море / город / двор" |

## Рекомендации

<Warning>
  Не создавайте отдельное поле для данных, которые уже есть в стандартных полях лота. Это усложнит фильтры, импорт и аналитику.
</Warning>

<Tip>
  Перед массовым импортом согласуйте список пользовательских полей с командой продаж и CRM-интегратором.
</Tip>
