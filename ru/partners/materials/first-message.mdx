Первое сообщение должно быть коротким, конкретным и без обещаний, которые не подтверждены командой GRIDIX.

## Шаблон для знакомого клиента

```text
Привет. Вспомнил про вас, потому что у GRIDIX есть решение для девелоперов: каталог проектов и лотов, заявки, CRM, виджет на сайт и брокерская сеть в одной системе.
Если актуально, могу познакомить с командой GRIDIX, чтобы они коротко показали сценарий под ваши проекты.
```

## Шаблон для делового контакта

```text
Здравствуйте. Хотел предложить познакомиться с GRIDIX.
Это платформа для девелоперов: управление проектами и лотами, публичный каталог, заявки, CRM, сайт-виджет и брокерская сеть.
Если у вас сейчас есть задача по цифровизации продаж или каталога объектов, могу организовать короткое знакомство с командой.
```
